# **App Name**: دفتر دي زاد

## Core Features:

- إدارة المخزون: إدارة تفاصيل المنتج والباركود وكمية المخزون وسعر التكلفة مقابل سعر البيع باستخدام Firestore.
- نظام نقاط البيع (POS): تسجيل معاملات المبيعات اليومية مع خيارات الدفع النقدي والائتماني وإدارة تحديثات Firestore.
- تتبع ديون العملاء (كرني دي كريدي): ربط المعاملات بالعملاء وتتبع إجمالي الديون وتسجيل المدفوعات الجزئية.
- لوحة معلومات التحليلات: تجميع بيانات الأرباح والإيرادات اليومية/الشهرية في تقارير مرئية في واجهة المستخدم.
- مزامنة البيانات في وضع عدم الاتصال: استخدام إمكانات Firestore في وضع عدم الاتصال لاستمرار البيانات ومزامنتها عند استعادة الاتصال.
- مصادقة المستخدم: تأمين التطبيق باستخدام مصادقة Firebase لتسجيل دخول المستخدم والتحكم في الوصول.

## Style Guidelines:

- اللون الأساسي: أزرق عميق (#3F51B5) من أجل الموثوقية والاستقرار، مما يعكس التركيز المالي للتطبيق.
- لون الخلفية: رمادي فاتح (#F5F5F5) للحصول على مظهر نظيف ومهني.
- لون التمييز: أصفر (#FFEB3B) لإبراز الإجراءات والمعلومات الهامة.
- خط الجسم والعنوان: 'PT Sans'، وهو خط إنساني sans-serif يجمع بين الحداثة والدفء.
- استخدام أيقونات تصميم المواد لتمثيل عناصر المخزون وأنواع المعاملات.
- تصميم تخطيط نظيف وبديهي يعمل على تحسين عرض البيانات وسهولة الاستخدام للمهام اليومية.
- رسوم متحركة دقيقة أثناء المعاملات ومزامنة البيانات.
